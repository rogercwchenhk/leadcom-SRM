# DESIGN.md

## 项目概述
AI驱动的采购管理系统，基于 Hermes Agent 实现智能需求理解、智能询价和辅助决策。

## 核心业务流程
1. 初步需求（内部员工发起，自然语言输入）
2. 需求确认（真实需求提出者确认，可与初步需求提出者为同一人）
3. 智能询价（Hermes Agent 自动发送询价单公开链接）
4. 比价分析（AI基于价格和货期分析）
5. 确定需求（采购专员确认）
6. 确定供应商（AI提供决策建议及选择理由）
7. 审批流程（超过金额阈值需一级审批，金额和审批人可配置）
8. 生成PO（采购合同）
9. 跟踪发货
10. 接受发票
11. 付款
12. 异常处理（退货、换货）

## 系统角色
- 初步需求提出者
- 真实需求提出者
- 采购专员
- 审批人员
- 公司财务
- 供应商

## AI功能 (Hermes Agent)
- 自然语言需求理解：从自然语言中提取产品名称、规格、数量、货期
- 智能询价自动发送：生成公开询价单链接，无需供应商登录
- 供应商选择决策建议：基于价格和货期分析，展示选择理由
- 全流程辅助决策

## 供应商协作界面
- 签署合同/订单
- 查找历史合同/订单
- 输入发货单号
- 提交发票照片
- 查询付款情况

## 技术栈
- Framework: Next.js 16 (App Router)
- Core: React 19
- Language: TypeScript 5
- UI: shadcn/ui + Tailwind CSS 4
- Database: Supabase
- AI: Hermes Agent

## 设计风格 - Anthropic 美学

### 视觉原则
- **简洁专业**：清晰的界面，去除不必要的装饰
- **柔和配色**：以中性色为主，点缀温和的强调色
- **充足留白**：呼吸感强的布局，元素间距合理
- **层次清晰**：明确的信息层级，引导用户视线

### 色彩系统
- **主色**：柔和的深灰色 (#1a1a1a) - 专业、稳重
- **强调色**：温和的橙色 (#f97316) - 温暖、活力
- **背景**：浅灰渐变 (从 #fafafa 到 #f5f5f5) - 柔和、舒适
- **卡片背景**：纯白色 (#ffffff) - 干净、清晰
- **文字**：深灰 (#374151) - 易读、专业

### 排版
- **标题**：Inter 字体，字重 600，字母间距略微收紧
- **正文**：Inter 字体，字重 400，行高舒适
- **数字**：等宽字体，增强数据可读性

### 动效
- **过渡**：200-300ms 平滑过渡
- **悬停**：微妙的上浮和阴影加深
- **加载**：优雅的脉冲动画

### 组件风格
- **卡片**：轻微圆角 (8px)，柔和阴影
- **按钮**：简洁设计，强调色填充
- **输入框**：细边框，聚焦时有柔和的光晕
- **表格**：斑马纹，悬停高亮

## 功能模块设计规范

### 合同管理模块
- **上传区域**：支持拖拽上传，边框虚线样式，hover时边框变色
- **PDF预览**：使用iframe或专业PDF预览组件，高度自适应
- **摘要展示**：分卡片展示关键信息，摘要文本使用柔和的灰色背景
- **状态标签**：使用不同颜色区分合同状态（processing-蓝色，summarized-绿色，linked-橙色）

### 需求管理模块
- **需求类型标识**：确定需求使用蓝色徽章，非确定需求使用橙色徽章
- **合同关联显示**：在需求卡片中展示关联合同的摘要信息
- **外销价格参考**：使用醒目的卡片展示，包含价格来源和置信度
- **询价对比**：表格形式展示多个供应商报价，支持排序和筛选

### 数据分析模块
- **图表配色**：使用柔和的渐变色系，避免刺眼的高饱和色
- **数据卡片**：大数字展示，配合趋势箭头（上涨-绿色，下降-红色）
- **时间选择器**：简洁的日期范围选择，支持快捷选项（本周/本月/本季度）
- **图表交互**：hover时显示详细数据tooltip，点击可钻取
