# DESIGN.md

## 项目概述
AI驱动的采购管理系统，基于 Hermes Agent 实现智能需求理解、智能询价和辅助决策。

## 核心业务流程
1. 初步需求（内部员工发起，自然语言输入）
2. 需求确认（真实需求提出者确认，可与初步需求提出者为同一人）
3. 智能询价（Hermes Agent 自动发送询价单公开链接）
4. 比价分析（AI基于价格和货期分析）
5. 确定需求（采购专员确认）
6. 确定供应商（AI提供决策建议及选择理由）
7. 审批流程（超过金额阈值需一级审批，金额和审批人可配置）
8. 生成PO（采购合同）
9. 跟踪发货
10. 接受发票
11. 付款
12. 异常处理（退货、换货）

## 系统角色
- 初步需求提出者
- 真实需求提出者
- 采购专员
- 审批人员
- 公司财务
- 供应商

## AI功能 (Hermes Agent)
- 自然语言需求理解：从自然语言中提取产品名称、规格、数量、货期
- 智能询价自动发送：生成公开询价单链接，无需供应商登录
- 供应商选择决策建议：基于价格和货期分析，展示选择理由
- 全流程辅助决策

## 供应商协作界面
- 签署合同/订单
- 查找历史合同/订单
- 输入发货单号
- 提交发票照片
- 查询付款情况

## 技术栈
- Framework: Next.js 16 (App Router)
- Core: React 19
- Language: TypeScript 5
- UI: shadcn/ui + Tailwind CSS 4
- Database: Supabase
- AI: Hermes Agent
